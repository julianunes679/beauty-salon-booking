# 💇‍♀️ Salão Bella — Projeto Web No-Code + Código

##  Sobre o projeto

Este projeto foi desenvolvido com o objetivo de criar uma solução digital simples para um pequeno negócio local (salão de beleza), utilizando ferramentas no-code e padrões fundamentais da web.

##  Tecnologias utilizadas

* Google Sites (no-code)
* HTML
* CSS
* JavaScript

##  Acesso ao projeto

* Site (no-code): https://sites.google.com/view/salaobella/in%C3%ADcio
* Repositório (código):https://sites.google.com/view/salaobella/in%C3%ADcio
* Sistema funcional: https://julianunes679.github.io/beauty-salon-booking/

##  Funcionalidades

* Página institucional do salão
* Exibição de serviços e preços
* Formulário de agendamento
* Interação com JavaScript (alerta de envio)

##  Aplicação dos padrões web

* HTML: estrutura da página
* CSS: estilização do formulário e botão
* JavaScript: interação com o usuário

##  Responsividade e acessibilidade

O site foi estruturado de forma simples e adaptável a diferentes dispositivos, com organização clara de conteúdo e navegação intuitiva.
